🎨 SVG Sprite Plugin 예제

이 예제는 vite-plugin-svg-sprite-builder의 사용법을 보여줍니다.

아이콘 목록

#icon-home
#icon-user
#icon-settings
#icon-heart

사용법

// vite.config.js import { svgSpritePlugin } from 'vite-plugin-svg-sprite-builder' export default { plugins: [ svgSpritePlugin({ iconDir: './icons', mode: 'inline' // 'inline' | 'file' | 'hybrid' }) ] } // main.js import 'virtual:svg-sprite' // HTML에서 사용 <svg class="icon"> <use href="#icon-home"></use> </svg>
